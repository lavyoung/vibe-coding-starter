# 行为准则

## 我们的承诺

我们希望本仓库成为一个对“文档优先 + 人类/AI 协作交付”感兴趣的人都能安心参与、理性讨论、共同改进的地方。

贡献者、维护者和使用者都应共同维护一个：

- 尊重他人
- 重视事实
- 乐于解释
- 愿意澄清上下文
- 对不确定性保持诚实

的协作环境。

## 我们鼓励的行为

包括但不限于：

- 清楚、具体地表达观点
- 反对一个想法时，聚焦问题本身，而不是攻击提出者
- 显式写出假设，而不是隐藏前提
- 欢迎新用户提问
- 把反馈集中在复用性、清晰度和正确性上
- 明确指出哪些内容是主观偏好、局部经验或尚未验证的判断

## 不可接受的行为

包括但不限于：

- 骚扰、辱骂或人身攻击
- 轻蔑、羞辱、嘲讽式表达
- 恶意争论或刻意带偏讨论
- 未经允许公开他人的敏感或私人信息
- 故意制造误导
- 把不真实或未验证的技术说法包装成事实

## 适用范围

本行为准则适用于以下场景：

- Issue 区
- Pull Request 区
- Discussions 讨论区
- 仓库评论区
- 其他由本仓库维护者组织和管理的沟通空间

## 执行方式

仓库维护者有权对违反本准则的评论、Issue、PR 或其他贡献进行编辑、隐藏、关闭、拒绝或移除。

对重复或严重违规行为，维护者可以采取临时或永久限制参与的措施。

## 举报方式

如果你发现违反本行为准则的行为，请通过仓库维护者预留的联系方式或仓库沟通渠道联系维护者。

## 额外说明

本仓库欢迎对工作流、文档结构、状态规则和协作方式进行强烈而诚实的分歧讨论。  
可以激烈反对观点，但不可以不尊重人。
