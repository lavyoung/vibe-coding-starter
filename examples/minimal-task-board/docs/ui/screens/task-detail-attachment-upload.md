# 任务详情附件上传场景（管任务详情页的上传入口、文案和状态，改界面规则时先查）

## 文档元数据

- 文档类型：ui-screen
- 当前状态：已落地
- 适用阶段：界面设计、联调、回顾
- 最近更新：2026-06-18

## 1. 入口

- 任务详情页右侧“附件”区域

## 2. 字段与控件

| 区域 | 字段 / 控件 | 说明 | 约束 |
|---|---|---|---|
| 附件表单 | `filename` | 附件显示名 | 必填 |
| 附件表单 | `url` | 附件访问地址 | 必填，URL 字符串 |
| 附件表单 | “添加附件”按钮 | 提交当前录入 | 单任务最多 3 个附件 |

## 3. 状态

- 成功反馈：附件列表立即展示新项
- 错误态：超过 3 个附件时提示“每个任务最多上传 3 个附件”
