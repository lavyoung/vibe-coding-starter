# V1.2.0 任务附件上传需求（管业务诉求与验收范围，开始设计前先查）

## 文档元数据

- 文档类型：requirement
- 当前状态：已接受
- 适用阶段：需求分析、设计收敛
- 最近更新：2026-06-18

## 1. 目标

任务详情页需要支持录入附件 URL，方便团队在任务中补充原型图、截图和说明材料。

## 2. 需求范围

- 每个任务最多添加 3 个附件
- 每个附件包含 `filename` 和 `url`
- URL 方式即可，不要求二进制上传

## 3. 验收标准

- 可以查询任务当前附件列表
- 可以新增附件
- 超过 3 个附件时应返回明确错误
