# RFC 目录说明（管重大技术/架构提案，方案未最终拍板前先查）

## 文档元数据

- 文档类型：rfc-process
- 当前状态：已生效
- 适用阶段：技术方案设计、架构评审、迁移设计
- 最近更新：YYYY-MM-DD

## 什么时候写 RFC

- 新增跨模块能力
- 引入或替换中间件、外部系统接入方式
- 调整模块边界、线程模型、锁模型、事务模型
- 存在两个及以上可行方案，需要显式比较 trade-off
