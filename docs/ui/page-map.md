- 文档类型：ui-page-map
- 当前状态：已生效

## 1. 这份文档解决什么问题

本文件用于维护 UI 层总入口，回答：

- 当前有哪些页面、弹窗、关键交互场景
- 哪些页面已经有单独的场景文档
- 哪些页面仍缺少设计沉淀

## 3. 维护规则

- 新增关键页面、上传入口、复杂弹窗时，优先补一条记录
- 若页面已成为开发依据，对应状态不得停留在“仅讨论”
- 若页面已下线或废弃，应同步更新状态或移除入口
